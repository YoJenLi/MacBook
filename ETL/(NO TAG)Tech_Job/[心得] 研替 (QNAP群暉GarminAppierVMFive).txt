群暉

跟 QNAP 不同，主打家用 NAS
因為爬文說會考 OS (像 RAID 之類的)，所以在去的路途中K完自己的OS筆記
結果唯一有準備的東西最後也沒考到 Orz

不過這家不像大家講的那麼恐怖啦，這家是我所有面試公司裡面考試最順的 xD
白板題後來問過有去面試的朋友，大家都不一樣，不過大多是基礎算法
如果不熟的話複習 Introduction to algorithm 並實際寫出來練手感
我認為就足夠應對了

我自己寫白板的流程
(1) 問清楚所有條件限制，並列在白板上
(2) 先簡單口頭講述你的做法
(3) 詢問自己能用到什麼程度的 library
(4) 定義自己的 data structure
(5) 定義 API
(6) 寫 code 的時候同時口述這一行或這個變數的作用，先聲明自已最後再考慮邊界測資
(7) 寫完之後，開始列幾個測資，從一般測資到邊際測資，來做細部修正驗證正確性
(8) 最後分析時間複雜度、空間複雜度，bottleneck在哪
(9) 說明在不同的限制條件下自己可以調整哪些部分達到不同的效能或省下多少記憶體

照這流程走寫完通常就是換下一題目，很難追問下去 xD
頂多給完時間最佳解後，再問你怎麼改能達到稍微慢一點但空間最佳解
不過中間有被面試官嫌棄說實務上不會像我白板那樣各種怪招優化，
因為可讀性差 (′・ω・‵ )

研替總共會分兩次面試，共四個人
前三個面試官會考白板題，跟問 Resume 的問題
最後一個面試官則是談 offer

(1) 群暉主要看 bonus，底薪的話真的偏低
(2) 之前沒有加班費，但說之後會有新制度
(3) 是 human resource pool，也就是只有進到公司之後才會分部門

Garmin

先考筆試，HR事前問說要考 C/C++ 或 Java
想說自己對 design pattern 沒很熟，不太敢說自己會 java
然後 C/C++ 至少還有 C，C++就放推

恩，結果只有 C++ 沒有考 C…..
而且實際上只有兩大題跟C++有關，剩下考的東西蠻雜的
像是給 design pattern 名詞叫你把code寫出來
給 java 現有的 api，問說這些 api 的漏洞及解決方法等問題
(像如果有 SQL injection 風險的話就要自己多做判斷之類的)
7大題裡面我只有3大題有把握是正確的，一半都不到 Q____Q

筆試完後
就做人格測驗，及模擬多益 (如果兩年內有考多益的話就附上證書即可免試英文)

面試會問 resume 的東西
後來自己乾脆把電腦拿出來，開 code 來解說 xD
中間一度被臨時要求要做英文面試 Orz
硬著頭皮把它給講完
整體來說面談很愉快

極力推薦 Garmin!!!!
(1) 他們非常重視每一個面試者
(2) 在這家公司工作能有生活品質，很吸引人
(3) 會有跟 global team合作的機會
(4) 最重要的是，會想在這個主管底下做事！ xD

Appier

因為聽聞跟 spark & hadoop相關就丟丟看
後來通知我說他們沒拿到研替名額
不過還是邀請我去面試，如果我是他們很想要的人，說不定會幫我保留一年offer

一開始就給一台公司電腦，當場上機考
連進一個網站，然後要求在網頁上的文字方塊寫code
像是 ACM 那樣，Submit之後會跑隱藏測資來測正確性

題目的話是基礎算法，面試官會多給你限制條件
應該是跟群暉一樣每次都隨機指定吧 xD
那時被問手刻 quick sort 但要限制記憶體用量
面試官說大家都給一小時，之前最快有人三十分鐘就寫完，應該很足夠

題目不難，但最大的問題是不能自己另外在電腦開檔生測資測試
當初寫完過範測，有三個隱藏測資吃了 segmentation fault
問了面試官說能否在電腦上另外開檔測試，不行 Q___Q
後來鐵了心直接用 online judge debug 法
發現原來是一個條件判斷漏打了一個 > 符號 xD

接下來大概問一下 resume
喔因為會有跟外國RD合作，所以面試官要求面試過程要英文面 Orz
然後開始寫白板問系統類的問題轟炸 Q___Q

例如說給 hadoop 中 runtime 某個地方異常得慢或資源用量異常，說明原因
基本上系統就是 A 出事，但原因通常是 C or D 設定值過大或過小
就是要從 D 開始說明，一路到影響的 C -> B，最後到結果 A

第二種就是給一個問題 (data scale 當然都很大)，然後資源有限制
問說怎麼解這問題

最後我有被問給一些條件下，試著去設計一個系統

整體來說我自認為表現不佳
主管最後點評說大方向都對，但經驗上些許不足
有被稱讚 code寫得很快 >/////<
最後邀請我進行跟 CTO 及 CEO 的二面
但要先回去請示說是否可能幫我保留一年正職offer

結果收到信說如果當完兵後還有興趣的話可以跟HR聯絡
看起來跟感謝函 90% 像 ( ˊAˋ)

不過主管真的很強 <(_ _)>
散發出的強者氣場會讓人想要在他底下做事 xD
這是我收穫最多的一次面試！
