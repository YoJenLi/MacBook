: 1. 鉅瞻 (地點：內湖)
: protocol組，考C(不難)
: 主管人不錯，講解很細心。
: 要看很多SPEC，不用靠寫code吃飯，但英文要OK，滿有機會出差。
: 八點左右下班。
: 走之前主管說之後面試的心得和他分享一下，一星期後通知結果。
: 回去後有通了幾次Email，主管也有回信。
: 結果：沒通知，寄信去要了張感謝函。
主管中意，大概是老闆不中意吧，老闆只重"資歷"。

: 2. 兆赫 (地點：中和)
: *面試過所有公司中感覺最糟糕的一間。
: 考C、資結、嵌入式系統(沒修過不會寫)。
: 很年輕的主管來了以後，請我自介及論文介紹，聽完後沒發表任何意見，
: 只"嗯"了一聲，就說他們是做set-top box的，講了一堆專有名詞簡稱的，
: 然後說我做的跟他們沒關，問我有什麼問題，
: 那今天就先這樣，(臉上寫著我很忙沒空理你)，結果三天內通知。
: 面談過程大概5分鐘。還好我騎車去大概10分鐘而已，不然真的還滿不爽的。
: 結果：沒通知。
業務型主管，假專業真外行越會講越不會做，他不中意你，大概是你不會講，
沒有讓他覺得你很專業。

: 3. 正文 (地點：中和)
: 考C、英文(仿多益)，不會很難。
: 兩個主管和我面試，部門是做VoIP的，主管說會找我去是因為跟我做得很有關聯，
: 不過我在學校做的只是皮毛，他們是做整個系統。
: 工作都在寫code，要非常有興趣才來，有興趣的話再寄信給他。
: 回去後隔兩天有寄信去表示很有意願，主管說還在挑人。
: 結果：沒通知。
先貶低你在學校做的東西，這類主管通常就是靠貶低別人混過來的。
又說寫CODE要非常有興趣，推測的出來，這工作很操，常常加班。
這間公司離職率應該很高，愛打壓別人，工作量又大，做越多罵越多。

: 4. 奇勤 (地點：板橋)
: 考C、邏輯測驗。
: 這間公司離我家超近，騎車5分鐘就到了，又在捷運站的樓上，
: 不過是間非常非常新的小公司，才剛成立沒兩個月(辦公室都還在裝潢，沒幾個人)。
: 主管說公司是由ASUS、台積、大小M 等等的資深主管聯合出資2億左右創立，
: 主要是做sensor的，產品預計明年中才會開始做。
: 結果：沒通知。
主管明顯是業務型的，小型公司最傷的就是沒專業主管，沒專業主管，千里馬也會
變成草泥馬。

: 5. 永碩 (地點：北投)
: 考C、Linux一些指令和觀念，都很基本。
: 主管人不錯，聽我自介和介紹論文完之後有稍微討論了一下，
: 然後介紹部門在做什麼(種類很多，記不得了)，
: 以及公司歷史(華碩為啥要分出和碩+永碩等等)，然後小小抱怨了一下。
: 幾乎都不用加班，大約六點大家就去吃飯了，然後會去運動，
: 等八點下班打卡(因為八點後下班晚餐才不用錢)。
: 結果兩周內通知。
: 結果：沒通知，打去問要了張感謝函。
這間感覺是不錯的公司，但是可能真的競爭太激烈了

: 6. 傳象 (地點：復興北路上)
: 沒有考試。
: 第一個面試我的是測試部門的，請我自介完後說我一開始可能不適合做測試，
: 於是請了軟體的主管來和我談。一進來遞了名片給我，發現是副總!
: 再次自介完了後，從我論文裡挑了很多細節的東西問，
: 同一個問題換了講法再問了好幾次。
: 問完以後說他們比較想找資工且有經驗的人(心裡OS：那找我面試是...)，
: 因為開公司是要賺錢，他們沒時間去訓練那些只是"會"寫程式的人，
: 現在和以前不一樣，以前公司願意花很多時間訓練你。
: 要很強的人才能幫公司賺錢，公司台清交碩的一大堆blablabla之類的，
: 又講了一堆大道理。
: 最後請了董事長來跟我簡單的面談。
: 董事長：你好，我是這間公司的老闆! 哈哈哈~
: 我    ：(馬上起身，大驚!!)老闆好，老闆好!!!  <-整個很像被驚嚇過度的白癡XD
: 結果：沒通知。
這間公司感覺是閑閑沒事做，無聊徵人過過當老闆主管的乾癮。看他們講一堆
五四三，看那些也都是靠打壓別人混過來的。
