分享一下我的，
38k (台商)->50k(外商)->xxk (外商) 保密一下

一開始在接案公司，32k，速度跟你差不多，約1個月做一支
做了快一年老闆良心發現，加到38k
做了一堆外包，UI很漂亮可是下載率都超低的，
有拿過排行榜第一名，但是咻一下兩個禮拜就下來了
別人的module包一包會動就好，甚麼都寫過甚麼都不是很熟

做得很膩跳到外商，維護使用者幾百萬的app
幾乎全部底層都自己寫，
C code寫得都快比objC還多XD

寫了一堆底層的東西，產出率卻越來越低，最近半年只寫了一支app
也練了一些跟寫iOS沒直接相關的，像是TDD,scrum,pair programming

這幾年寫iOS人越來越多，這是好事吧，
表示市場還是很需要寫mobile的人，有經驗的人才越能展現價值

薪水的話，大方向是國外>>>外商 > 台商
強烈建議找外商，除了比較敢給以外老闆也比較有腦

※ 引述《shinrenpan (iOS5)》之銘言：
: 分享一下我的 iOS 歷程
: 民國98年去資X會學 mobile 課程, 當初教的是cocos2d
: 學成後臺北找了一圈, 最高開到 28k, 所以我跑去竹科
: 竹科不成文規定就是大學沒經驗32k, 接來下跳公司的薪資如下
: 32k(竹科) -> 40k -> 42k -> 50k ->55k
: 基本上在業界, 90%注重的是 UI 客製化及外包接案
: 在某家公司, 我基本上1個月產一隻 App, 所有 UI 大概都是客制的
: 你不要想用 UITabBarController, UINavigation push pop,
: PM不知道這種鬼東西, 只會跟你說某 App 有這功能, 你做吧
: 用了內建的原件, 基本上很難做出來
: 所以業界你只要會刻很複雜的 UI 就會過得很好了 XD
: 不過刻 UI 到一段時間, 你會覺得自己沒進步, 而且公司不會允許你用新技術
: 例如 iOS7 某些API 你必須等到 iOS8 出才能用, 因為你開發的 APP 需要支援 iOS6
: 例如 passbook 跟 iBeacon, 我只能自己摸摸, 公司永遠不會用的XD
: 至於薪水跟投履歷, 我的104簡介只附了自己的部落格 URL, 介紹我開發過的 APP
: 基本上我都會將開發過的 APP 錄影模擬器跑的畫面, 打開履歷就會收到很多面試
: 薪水的話, 你開發過那麼多 APP 至少要開 40k 以上吧, 不過這兩年 iOS 也不太好找了
