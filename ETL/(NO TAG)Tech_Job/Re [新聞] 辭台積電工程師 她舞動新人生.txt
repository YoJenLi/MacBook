: 惜放棄飯店接班的計畫。新竹市也有一位勇於追尋舞蹈夢想的女孩，她不僅擁有清、交大
: 高學歷，還是台積電的工程師，仍辭掉人人欽羡的高薪工作，跑去跳肚皮舞，勇氣可嘉。
:
: 卅二歲錢怡，綽號Money，漂亮書也念得好。交大應用數學系畢業後，攻讀清大統計研究

: 所，畢業後在台積電擔任工程師五年。三年前辭掉工作，到台北極緻國際舞團擔任舞者，
: 專攻爵士舞、肚皮舞、流行舞蹈及鋼管舞。
:
: 錢怡念清交大時是熱舞社成員，特別喜愛街舞。或許是為了滿足父母期待，所以畢業後和
: 很多的清交大校友一樣，到台積電等知名企業工作，雖然可以賺到不少錢，卻發現這不是
: 她的興趣。
:
: 錢怡說，快卅歲時，她決定勇敢追逐夢想，「如果不做，這輩子恐怕再沒有機會了！」她
: 把想法告訴父母，父母嚇了一跳，堅決反對。
:
: 錢怡就像《舞力全開4 3D》電影的女主角一樣，為了要當專業舞者，不惜辭掉工作，專心
: 從事最有興趣的舞蹈工作。經過兩年的努力，錢怡不僅成為優秀的舞蹈老師，更先後得到
: 亞洲肚皮舞節暨第四屆世界肚皮舞國際邀請賽國際個人組二等獎、台中市議長盃全國創意
: 肚皮舞公開賽團體組冠軍等多項頭銜。
:
: 錢怡說，跳舞絕不是低俗，反而可帶給她無比的快樂與開心，就像電影中的台詞，「如果
: 人生不敢突破現狀，有何意義？」她認為，人生如果有夢，一定要勇敢追尋，就算失敗，
: 也沒有遺憾了。
