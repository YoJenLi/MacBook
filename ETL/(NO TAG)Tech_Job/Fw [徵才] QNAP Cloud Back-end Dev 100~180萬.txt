你對打造自有品牌的雲端服務有夢想嗎？
一身功夫只能去幫外國人或強國人打工嗎？
還是在做白牌設備嗎？做服務總是只支援繁簡中兩國語系嗎？

不要再遲疑，如果你跟我們一樣也希望以台灣之名做出全世界都喜歡的服務，請加入我們！


< 我們做的事 >
* 現在: 提供雲端服務讓使用者可以既方便又安全的遠端存取與分享NAS資料與服務
    入口網站: https://www.myqnapcloud.com

* 未來: 打造全方位的資料管理雲服務，讓資料可以在您與朋友的多個空間無礙的流動
    服務上線可推廣到全球超過百萬的消費者，不怕做出來沒人知道沒人用


< 我們能提供的 >
* 業界中上package: 100萬~180萬/年 (看需求符合程度)
* 績效導向，沒有不健康加班 (萬一需要可遠端連線繼續努力)
* 黃金地點 (捷運忠孝新生站出口前, 下雨天不用帶傘)
* MBP + 大螢幕 (可再+1)
* 有機會貢獻open source專案
* 打造國際級軟體服務的機會
* 擁抱Agile與DevOps等先進精神
* 員購NAS (約6折)


< 我們的需求 (按比例分配) >
< 硬技術 >
* (25%) 開發大型Distributed System經驗(考量High Availability, Scalability, Load Balance), user愈多, 流量愈大, 邏輯愈複雜愈好.
* (20%) 熟Java, Python, Golang等可用來開發大型系統的OO語言, 熟functional語言加分.
* (15%) 熟Distributed System Infra. 如: NoSQL, Distributed Cache, Message Queue, Distributed  Coordination, Event Bus, IaaS, PaaS.
* (15%) 熟悉軟體開發每個環節: User story, System/architecture design, Implementation, Testing, CI, Deploy, Operation.
* (10%) 貢獻open source專案的經驗, user愈多愈好
* (10%) 對技術細節有潔癖. 每看到一次bug就會想用不同方法/設計/工具來避免再度發生.
* (5%) 持續追蹤新技術與其適合的scenario. 如: docker, spark, micro service, web assembly.

< 軟技術 >
* (30%) 團隊 > 個人的態度, 主動幫助別人, 如Scrum精神持續改進, 分享知識, 帶給全團隊+10戰鬥力的靈氣
* (30%) 溝通: 有效率的善用各種溝通方式, 撰寫文件, 報告/demo (筆試要用英文寫喔~)
* (15%) work smart > work hard, 以正常工時為第一目標
* (15%) 帶領團隊的經驗, 如何打造軟體人才想久留的環境
* (10%) 說服我們您的跨界興趣/才藝對工作很有幫助, 如: 旅遊達人, 星海2大師, 跳舞狂
    我們希望團隊組成盡量多元一點, 一起激發更多創意!
