  總要晚上幫忙換尿布，哄小孩睡覺和接送小孩，難道要丟給老婆?

2.孩子的人生只有一個，賺那麼多錢，以後小孩子的記憶裡沒有爸爸，錢賺那麼多
  意義也不大，除非你像賈伯斯一樣，覺得事業比孩子重要

3.回過頭來看，舊的公司已經虧損，ㄨㄟ謀ㄟ事情越來越多了
  也不是你能解決的，你也被空降部隊壓過去了，該是離開的時候

4.不囉嗦，就直接選新公司了，全球50大，沒啥好挑的了。
  能在全球五十大，夢寐以求。

5.外商的好，待過才知道，超爽der~

: 大家好
: 最近又面臨了一個工作抉擇，希望分享一下
: 聽看看同是科技業的大家會怎樣看待我面臨的這個選擇
: 如果有類似的經驗，也歡迎先進們一起分享
: 工作7年了，剛出社會時，沒什麼目標
: 所以跟很多人一樣只知道往GG跑
: 做了兩年，雖然老闆很賞識，也有不錯的考績與分紅
: 但實在是沒辦法接受FAB的生活所以選擇離開了
: 離職時接受了老闆的建議，到了一家準備上市櫃的中小型公司
: 沒想到這一進就待了4年多，這4年間過得很精彩
: 中小公司因為資源有限，所以必須身兼RD、PE、CSE、FAE、PM甚至QA
: 從ISO的制定到產線的規劃、量產以及客戶服務甚至產品行銷及訂價和Audit通通都包
: 每年要飛美、日、韓、歐洲等十多個城市與主要的幾大半導體公司討論新產品的開發
: 大家可能無法相信為什麼這些工作會交給一個年輕人
: 但的確如此，因為董事長很年輕
: 是一個肯給年輕人機會衝刺的領導者
: 四年下來，每年有1/3待在國外，2/3待在國內，平均工時>15小時&日夜顛倒
: 我也的確做出了一些成績，雖然很累...但公司也沒虧待我
: 給了股票、也給了我一個team讓我有機會當中階主管
: 還有一些機會參加台、政的MBA課程，學習一些商業知識
: 一切看起來相當的完美....
: 然而
: 隨著公司上市、櫃之後，很多決策因為市場壓力不得不向投資人妥協
: 高階管理層來了一堆空降部隊，雖然很快有漂亮的財務表現
: 但很多資源開始導向了短期可獲利的專案(ex.OEM)
: 出現了很多業內/業外的投資及合併等財務操作也積極的擴建廠房增添固定資產
: 一些長期才能回收卻重要的專案資源卻因此被稀釋
: 可預見的問題很快的在今年初出現了，公司開始虧損
: 預算、資源不斷限縮、鬥爭及權謀不斷上演.....
: 這個狀況整年不見好轉，甚至越來越惡化
: 所以我向公司表達了辭意
: 但因董事長待我不薄，人情讓我面臨艱困的選擇
: 這兩年一直有獵人頭公司找上門
: 這樣的環境下，於今年Q3我接下了一份offer
: 這是一間全球50大的公司，給了一個基層主管的位置
: 一樣是新產品開發的國際性工作職位
: 因為工作分工較細，全球也有八十多個支援辦公室
: 所以能有相對較為正常的工時
: 當然也有一些挑戰性，需要直接向總部的老外report
: 因為我就要結婚生小孩了，很難繼續當空中飛人，所以我心動了
: 一邊有董事長這個貴人的人情債，一邊是人人稱羨的百年大公司
: 一邊有爆發性成長的機會，一邊能有安安穩穩的產業龍頭地位
: btw，兩邊給我的待遇以年薪而言並不會差異很大
: 如果是各位先進...你們會怎麼看待這個選擇???
: 會繼續賭嗎.....
