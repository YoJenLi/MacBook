Works Applications
這是從板上的徵才文應徵的
說明會聽到一定要先去上海的時候本來想放棄
可是還是決定考考看，想說考得不錯的話也許有籌碼可以試著凹去東京
第一階段是個程式測驗，兩個題目 C++ 跟 Java，五天時間寫
C++ 算是經典的題目，Java 我是看不太出來想問啥
deadline 後沒幾天就收到面試通知
面試當天第一關對工程師主管，講英文，簡單自我介紹後直接上機考
我記得大概四或五題，應該是看答題狀況決定接下來的題目，語言沒記錯是不限
不過過程中工程師沒有討論，只有我跟他確認需求，還有答完後他再追加條件
第二關是筆試，寫一些國高中程度的數學題目
第三關對人事主管，只會日文，可是有帶一位中文翻譯
我大部分是用日文對答，覺得比較重要怕誤會的問題才說中文請他翻譯
結束後等了一下就發 offer 了
不過當時我說如果要去上海的話，我需要跟家人慎重討論才能決定
之後過了一個多禮拜，正當我準備寫信婉拒的時候
對方先寫信來了，說他們公司正在討論讓我直接到東京，要我稍等一下
大約三個禮拜後再度通知說可以去東京了!!不過要再重新面試一次 XD
這次是走中途採用的管道，要全程用日文，包含聯絡的信件
於是就安排了跟 recruiter 的 Skype 面試
那時候一方面第一次在正式場合說日文
二方面 recruiter 問的問題滿刁鑽的，說實話我有點招架不住
比方說他問你不想去上海對吧
但就算你來東京了，可是公司又把你派去上海的話怎麼辦
我當下只能傻笑，所以結束後覺得大概沒救了
沒想到兩個禮拜後突然收到通知說通過了，要約 onsite interview 的時間(去日本)
在日本面試的時候也是先對工程師主管
但是這次沒有上機考(他是說我先前考得不錯所以沒必要)，直接開始聊天
不過面談中我的對答很多矛盾的地方，結束後就被人資帶下樓了
沒談到人事主管，果然兩天後就收到不錄取通知

Indeed
這間其實我沒有進行到很後面
不過因為我透過不同管道應徵了兩次，覺得滿有趣的所以和大家分享
第一次是板上的徵才文
他們先舉辦了一次程式競賽，我記得大概拿 150 位左右
兩個禮拜後通知說有敗部復活的機會，可是我弄錯時間所以就沒了 orz
第二次是跟東京 Works Applications Skype 面試結束後收到一封信
竟然是獵人透過 LinkedIn 來介紹日本的工作機會
由於當時處於無望的狀態，看到獵人的邀請真感覺上帝又開了另一扇窗 XD
而且好巧不巧又是 Indeed
第一階段也是程式測驗，模式跟上次的程式競賽一樣
低空飛過後安排了 Skype 面試，對工程師，講英文
他先介紹他們公司跟部門的工作內容，然後我自我介紹
之後開線上文件要我寫一個小程式
我是不覺得有那裡寫錯啦，但是隔天就收到感謝函了

Google
獵人介紹 Indeed 的時候一開始只說是一家 multinational search engine company
這才讓我想起來 Google 好像也可以投投看
於是就順便抱著聊勝於無的心態投了日本 Google 的軟體工程師
其實我去年也投過台灣的，但是石沉大海，沒想到這次隔天 recruiter 就找上門來了
recruiter 先打電話來聊天，主要是了解背景然後安排 phone interview
我參加兩次 phone interview，都是對工程師
內容也都差不多，先問一些電腦的知識，然後開 Google Doc 寫程式
通過後 recruiter 安排 onsite interview
因為我是台灣人所以是去他們的台北辦公室
當天要面對五位工程師的車輪戰，一輪 45 分鐘
我遇到的四位是台灣人講中文，一位美國(?)人講英文
一樣是我簡單自我介紹，他們會針對學經歷問一些問題，接著進行程式測驗
比較特別的地方是程式都要用紙筆寫
因為很重要所以再說一次，不是寫白板而是用紙筆寫!!
他們要收回去當作評分依據
我是都寫完整的 C++ code
不過當時忘了問是要寫可以跑的 code 還是 pseudo code 就好
測驗過程中他們問很多優缺點的分析，而且一個題目會有很多後續變化
其實都是考演算法跟資料結構的複合應用
但我實在未夠班，一個禮拜後通知說到此為止了

Synology 群暉
接二連三的失敗後，其實心情還沒調整回來
但是時間也有點晚了(當時已經是過年後)，只好硬上開始找台灣的工作
這間我透過同學投履歷，隔天就安排面試
面試當天大概就像大家說的一樣，過三個工程師 + 一個主管
面對工程師就是自我介紹後開始寫白板，主管來就是談 offer 了

Trend Micro 趨勢
這間是用 104 投履歷，很可惜他們過了兩個禮拜才看到
我是面試完 syno 出來，開手機剛好收到趨勢的面試邀請
當天先寫兩份考卷 C/C++ 跟 Java
然後有七個部門分成三組輪流來面試
流程也都差不多，我自我介紹、主管問問題、主管介紹部門，我問問題
最後人資來了解面試的感想，以及有沒有偏好的部門

NVIDIA
這是人資透過 LinkedIn 找上門
當天先寫一份考卷關於 Java 跟 Android
然後有人進來進行簡單面談，接著設定電腦開始上機考
這上機考比較特別，是要從 log debug，找出有問題的模組
上機考的時間一直到他們下班前，不過中途會有一些部門主管進來面談
結果是人生第一張無聲卡
附帶一提當天接洽的人資滿可愛的 XD

樂天
這也是透過板上的徵才文
第一階段是程式測驗跟 video interview，要回答五個問題並錄影上傳，都是英文
程式測驗題目滿簡單的，可是當下我就是一直沒辦法寫到完全正確
結果晚上睡覺一躺上床馬上跳起來，幹，想到了
之後我參加三次 Skype interview
第一次是對工程師跟 leader
一開始是用英文，不過對方看到我履歷寫會日文就問可不可以用日文進行
之後就完全用日文了
第二次是對另一個 leader
一開始也是說英文，他也有問我會不會日文，不過只用日文閒聊幾句就繼續用英文
第三次是跟主管，印象中他一開始就直接用日文了
前兩次以聊天為主，我的自我介紹跟他們介紹部門業務
第三次主要是主管介紹公司的文化
通過後因為我之前沒有考過大型的英文測驗
所以他們要求我考一個 Versant English Test
最後人資會有個 offer meeting 說明 offer 內容以及一些福利、補助
不過他說即使我過了 Versant English Test，入社後還是要再考多益...
