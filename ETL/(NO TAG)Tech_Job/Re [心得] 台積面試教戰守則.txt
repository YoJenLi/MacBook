: 想分享給大家 如何面試上台積
: 我的方法不確定能不能套用在我身上
: 但是我確實拿出成果
: 面試完2個working day後就拿到口頭offer
: rule 1 :
: 對你要面試的職缺了解  越了解好!!  因為這樣你可以問越多關於該職缺的問題
: rule 2 :
: 專心聆聽面試官所講的 , 並針對他們講的去從中發問 , 讓他們知道你有在認真聽
: 而且有在思考
: rule 3 :
: 拿出誠信!! 不要任何隱瞞 譬如我已經在工作 而且是新人 , 但還是老實說
: 撒謊 被發現就完了  誠信很重要
: rule 4 :
: 志同道合! 相信有投gg的一定都知道 志同道合 這四個字  面試我的HR就有提到此事
: 我就說 到底什麼是志同道合 (讓HR覺得你積極想了解該公司)
: rule 5 :
: 不要急著收尾 , 當主管或人資問你還有沒有問題 , 盡量不要有種要解脫的感覺
: 而說出:沒有問題 (讓對方覺得你有亟欲逃離的感覺)
: 結語 :  我不是什麼面試高手  我也是慢慢累積經驗的
: 想當初第一次面試U的時候 , 主管問我為什麼想來這個職缺
: 我還傻傻的說 : 因為你們找我來面試這個職缺阿...(我OS:我沒投履歷)
: 第一次面試GG的時候 還跟主管說 指導教授的壞話...結果連人資都沒面試到
: 現在想起來 當時真的有夠蠢的
: 不過真的是隨著面試經驗累積 會愈來愈好的
: 一次不進GG 不代表不會有機會進去  還是有可能有面試機會的
: 預祝大家進入 理想公司


我是TSMC老菜鳥，

rule1~rule5，

若拿給一個正常人看，

應該都判斷的出不管面試哪家公司都是基本準則，

rule1:誰不會特地去了解自己要去面試的公司?
rule2:誰面試會白目不認真面試官說話？
rule3:哪家公司發現你沒誠信還錄取你？
rule4:哪家公司不想找所謂志同道合的人？
rule5:論點正確，但面試哪家公司會有其他作法？

那至於原PO為什麼會發這篇面試教學文？

主要目的有三：

第一、媽，你的好兒子我上台積電了！！

第二、科技版其他公司或是還沒台積面試機會的魯蛇，我先上台積電了！！

第三、我年薪百萬，我是台積電工程師！

有沒有必要這麼昭告天下阿？
