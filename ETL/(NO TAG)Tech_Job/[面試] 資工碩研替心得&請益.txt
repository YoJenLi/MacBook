1. 晨星半導體 (軟體設計工程師)

人資很早就打來請我去面試

準備資料 :
    要準備20分鐘的投影片 介紹自己的論文
    但我論文還沒開始實作 所以投影片後面就放自己修課的作品
    人資說104有履歷就不用印了 但我覺得還是帶一下比較好
    我沒有帶 主管有時候沒東西可以看

面試流程 :
    一開始會考c/c++ 跟網路找到的考古題幾乎一樣
    不過我有同學寫到跟我完全不一樣的題目 可能要看哪個部門出的考題

    之後有三個部門4個主管來面試 首先要請我自我介紹20分鐘
    可是我沒準備那麼久 腦袋也擠不出東西
    大概介紹10分鐘不到就結束了= =

    接著投影片:論文介紹跟作品集介紹 感覺主管對這些東西都沒什麼興趣
    因為我修過計算機圖學 問了一些圖學的問題
        解釋圖學Pipeline  ray tracing在什麼座標系統算hit object..

    到一段落主管們就去外面討論 之後只有一個部門走進來
    介紹了部門 問我介不介意工時 出差 覺得自己耐不耐操

    主管說他們平均9點~10點下班 趕project才會到11點 (dtv部門)
    在弄graphic 比較底層的driver 能力比較強後會去韓國出差

結果 :
offer get



2.旺宏電子

一開始要聽演講跟考英文
之後就在會議廳等著 輪到你人資會叫你去餐廳面談。
面談很輕鬆 主管會請你先自我介紹 之後就像聊天一樣。
我跟主管聊到 旺宏工時蠻短的很棒。
但主管跟說年輕人不要怕操 要趁年輕多操操自己 10點下班OK的= =
他建議我去別家公司 不要去旺宏XDDD

結果 : 無聲卡Q_Q




3.聯發科技 (軟韌體開發工程師 & 多媒體演算法開發工程師)

有朋友的學長幫忙內投

準備資料 :
    個人履歷 + 論文大鋼 + 作品集
    論文大鋼我直接把準備給晨星的投影片印出來

面試流程 :
    當天有兩場 早上場跟下午場 首先要考c

    我看聯發科總共寄5封面試通知給我 以為只有5個部門。
    沒想到早上就有6個部門了 所以論文大鋼印不夠
    主管直接請我去白板說明論文 也問了許多論文的問題

    還有會看成績單有什麼科目問一下
    例如我有修訊號與系統 要我解釋fourier transform跟取樣定理
            修過多媒體資訊系統 問什麼是h.264
            問我電腦安全這門課有什麼印象深刻的... 還有很多

    接下來就是os轟炸了 面試前一定要好好準備
    網路上也蠻多題目可以看的
    計算機組織的pipeline也要記得5個stage在幹嗎
    以及為什麼要pipeline 跟hazard

    履歷上最後那五題也會問
    哪個bug最映像深刻 寫過最大的程式是什麼 co-worker經驗跟分工..等等

    最後部門介紹
    不過有點多 很難記起來每個部門在幹嗎

    早上面完之後非常累 還要挑戰下午場
    除了5位左右的主管 還有兩個視訊的 台北跟竹北
    竹北的有請我解釋網路7層 udp跟tcp差別

    下午場剛講完論文我就直接被殺爆  問一堆問題答不出來
    覺得自己表現相當不好

二面 :
    隔三天就收到二面通知
    有四個主管  都是聊人個特質
    問一些生活跟課業上的經驗 沒有專業科目

    主管好象都希望你有修過embedding system


結果 : offer get  但好像是沒有二面的部門


4.瑞昱半導體

準備資料 :
        履歷 + 論文大鋼 + 作品集

數位ic設計(TV) : 一天三場

   這個不知道事我投錯還是人資幫我排錯了
   主管們表明要有電機相關背景知識才可以進去

   不過主管還是有請我介紹論文
   而且都聽得懂  我覺得很厲害 因為不同領域

   第一場的主管直接問我有沒有上PTT
   他說:那你應該有聽過 上輩子做壞事 這輩子做電視吧XDDDD
   平均七點下班

   第二場是音訊的 他看到我有修訊號與系統
   考了我DFT跟convolution以及取樣定理

電腦周邊 演算法開發 : 這個才剛面完

   他們在弄音訊的演算法
   因為我碩論比較偏影像處理，本來希望是多媒體來面我的= =
   所以主管幾乎都在介紹他們部門 沒怎麼問我問題
   主管覺得 影象跟聲音都是訊號 所以懂影象跨聲音應該不會很吃力(?)

   他有出一個白板提
   intput  double* A,int N ,double x; //A就是系數ARRAY
   請你寫個function算出下列function
   f(x) = a0 + a1x +..+anx^n

   因為我跟主管說我數學好 又修過訓號與系統
   主管就跟我聊了很多  說工程師分成兩種
   一種想演算法 一種implement的
   想演算法需要靈感 所以不會加班太晚XD
   另一種可能有客戶壓力 就要加班
   他問我比較喜歡哪種XD
   主管說他們部門平均7點 大概一星期三天會到8點 兩天六點

   主管說這次面試他感覺蠻不錯的 所以我覺得我會拿到offerXD

5. 華碩 (軟體研發工程師(影像技術) 跟 影像處理/軟體開發工程)

準備資料 :
        履歷 + 論文大鋼 + 作品集

面試流程 :
   爬文好象是一開始會邏輯測線 考專業科目(一張考卷有C/C++ 資料結構 計組 OS)
   之後會跟人資面談

   邏輯測驗 : 四個圖案推出第五個圖案
   專業科目 : 選擇題 計組有考memory位置 這個我不懂他在問什麼
              其他都不難

   但是我沒有跟人資談就開始面試了

影像處理/軟體開發 :
   自我介紹完 直接介紹論文了
   我想說都是弄影象的 碩論就直接講方法 沒有從頭開始講
   但是這樣搞得主管們聽不太懂我在弄什麼
   我也不清楚他們不懂在哪裡
   覺得面試下來蠻糟糕的

   後來我問薪水 他們很驚訝人資沒有先跟我談
   主管說平均七點下班

軟體研發工程師 :
   這個是團體面試 我旁邊還有一個台大機械的
   主管一開始看看我們兩個的履歷 請我們稍為解釋一下論文
   再來部門介紹 讓你問問題 之後就發白紙考試

   總共考4題 比較算數學機智問答 有點像 組和數學 這門課
   考完之後 他會看你是什麼科系在出題目給你
   我又被問到取樣定裡跟 convolution
   還有給一個很長的算式 轉成postfix

   機械系的被問到 為什麼直升機要有尾翼
   怎麼投下墜球  兩個船一起開為什麼會相吸
   機械系的題目我剛好都會 好想替他答XDDDD

   最後主管會請你講 你在面試看到自己的優缺點

   一個月後等通知
   但主管說你把華碩放第一志願 而且又在他list裡面
   可以跟人資說 他會直接給你offer

   主管說平均8點下班 看自己進度
   他說你如果成為團隊中top的話 薪水可以比mtk高
   還說華碩其實薪水不會輸mtk很多 XD
