Insowe 暑期實習計畫

【計畫介紹】

Insowe 即將開始今年度暑期實習計畫，我們非常願意與實習生一起共事與成長。

在 Insowe 實習計畫中，你會有 Mentor 帶領指導，同時會被賦予很多發揮的空間，

我們也會適時的給予你工作、求學等各種建議！若你已經有大公司的實習經驗，那很棒，
但如果你想嘗試一下 Startup 的風格，JOIN US！讓你在求學過程中，有機會參與最實戰
的第一線戰鬥位置！若你還沒有實習經驗，這次的實習之旅定會帶給你非常不一樣的體驗
！

不會叫你(妳)倒水、掃地...等雜事，有行政助理我們會讓助理做，沒有助理也是創業的
我們來做，當然，如果是寫開發相關的文件那種和職務有關的工作，那不能算雜事。

雖然實習津貼比不上大公司，但是不讓實習生打雜，算是我們的用人誠意。

自 2013 年起開始的暑期實習計劃，現已有 1 位應屆畢業的同學，即將成為我們的正職
夥伴；有 3 位還留在南部的同學，成為我們的兼職夥伴。

我們是新創軟體公司，實習職缺就是要借助同學的專長，也提供同學一個職場歷練的機會
。

【團隊介紹】

Insowe 以優先的技術能力與國內數個 Startup 團隊合作，協助其短期內的 Prototype
上線或協助階段性任務，補足其不足的開發能量，藉此達成雙贏的成效！

對我們有興趣瞭解，亦可參考相關簡報：
http://www.slideshare.net/insowe/

本團隊是一個年輕有活力的新創公司，我們的產品和服務具有獨特性，十分適合喜歡挑戰
、想累積實戰經驗的你。

【時間地點】

自 2015 年 7 月開始，實習時段可討論，能全時段實習者尤佳！
工作地點在台南辦公室（台南大學附近）。

【實習鼓勵】

在這裡，實習生不是員工，可以玩在線專案，但搞砸不用擔責任，技術不會可以從頭教
為了鼓勵同學們認真學東西，以下是我們提供的一點鼓勵

實習獎金：預知詳情請面議
一份正餐：我們每天提供一餐！午餐晚餐都可以
實戰經驗：即時在南部，也不怕求學期間沒有實戰的工作經驗！

【實習夥伴】

年齡不限，優先考慮大二～研究所的同學

一、開發（共 5 名，各種類不限名額）

1.前端工程師/設計師

        不會可以從頭教，有興趣即可
        加分：熟悉 HTML / CSS / JavaScript(只會 jQuery 也沒關係）
        大加分：熟悉 HTML5 /CSS3 / SCSS / RWD 開發
        工作內容：前端頁面切版、製作

2.PHP 後端工程師

        不會可以從頭教，有興趣即可
        加分：熟悉 PHP / CI / YII / MVC 架構
        工作內容：開發企業用軟體的部分功能、開發 API 介面

【招募流程】

可用你善於呈現自我優點的方式（格式不限）附上相關檔案，等候我們進一步的回覆。
通過書面篩選的實習候選人將通知電話口試、面試等進一步通知。
