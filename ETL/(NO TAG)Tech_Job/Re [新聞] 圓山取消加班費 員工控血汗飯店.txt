有些人問說為什麼圓山飯店的勞資爭議要PO在Tech_job版

這是因為假如科技業的勞工不關心其他產業勞工的權益

那科技業的勞工落難時 其他產業的勞工也不會來關心

像之前的華隆罷工 在本版相對的少人關心

紡織業的中高齡失業勞工死活 似乎是另一個世界的事情?

餐飲業的超時過勞員工 似乎都永遠不會跟本版有關？

那等到真正"有關"的時後 就已經來不及了！！

小弟我作為一個當初躺在華隆頭份廠門口肉身擋 陳明朝/翁大銘 貨車的人

發覺的確不少科技業的勞工持這樣的態度:

"如果有能力 為什麼會失業?" "如果沒能力換工作 最後領不到退休金 是不是該怪自己"

台灣的勞工就是這樣被分化的 很多我們今天有的勞動權益其實就是這些當年前輩爭取來的

這讓我想到2008金融海嘯的時後 一些在其他版面上的冷言冷語

"阿不是科技新貴？不是以前賺很多？還有股票嗎？"
"能力強怎麼會放無薪假？能力強怎麼會過勞死？"
"什麼時候要去考公務員?"

如果"科技業某大公司"這層外皮只有在過年跟聯誼的時後有用

卻會阻礙我們觀看這世界的眼界 丟了他吧

不要停止關心身邊的人 因為可能某一天 你會需要他來救你

: 〔自由時報記者林嘉琪／台北報導〕有六十一年歷史的圓山飯店，遭員工抗議是「血汗飯
: 店」！
: 圓山飯店工會昨日抗議資方更改勞動條件、剝削員工，有員工上班十六小時卻沒給加班費
: ，更因人力不足導致員工常無法放假，甚至有員工已經十幾年無法回家吃團圓飯。工會強
: 調，若資方沒給滿意安排，下一步打算發動全台罷工。
: 圓山：加班改為補休
: 圓山大飯店工會理事長張舉成表示，圓山飯店在七十八年曾跟工會簽約，只要員工在國定
: 假日或例假日上班，就能領二倍薪水，但圓山飯店九十七年之後，取消給加班費，改為補
: 假時數。
: 人力少 員工補不到假
: 勞動條件沒有過去好，但工作量卻大增，張舉成舉例，圓山飯店常接待大批中客團，飯店
: 人力不足，無法放假，遇到工作忙不完還要銷假上班，累積補假是常見情況；有飯店清潔
: 工上班時間長達十六個小時，超過勞基法規定十四個小時上限，但資方還是不發加班費。
: 有資深員工表示：「年夜飯是飯店最忙時段，已經十幾年沒吃過團圓飯了。」
: 圓山飯店則表示，圓山工作人員平均工作年資高居全國五星級飯店第一，去年難得轉虧為
: 盈，但因立法院預算中心及董事會要求飯店必須持續降低人事成本，因此檢討七十八年時
: 訂的團體協約，將加班費改為補假，飯店為永續經營，必須做出一些改革。
: 圓山公關經理陳愷璜回應，目前就是依照勞基法，排班制員工假日上班不需發給加班費，
: 但可以在其他天安排補休。
: 陳愷璜並表示，去年每位圓山飯店員工可領二．五個月年終獎金、○．一到○．五個月年
: 資酬勞金、三萬四千元激勵金、二百到三百元春節上班紅包、三千元春節紅包、每年二百
: 元年資加給，員工福利比同業優渥。
: 但張舉成回應表示，基層員工平均月薪才二．五萬元左右，加上加班費大減，員工勞動條
: 件明顯變差，圓山近來又花大錢進行整修，耗資費用超過營業所得，要求資方落實團體契
: 約、總經理應由內部升遷、董事會應有勞工代表、退休準備金要存入台灣銀行帳號，並要
: 求撤換董事長等。
: 圓山飯店人員表示，對於工會仍不滿意的部分，勞資雙方會再繼續協商。
