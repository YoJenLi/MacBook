※ 引述《crave (白雲黃葉共悠悠)》之銘言：
: 如題
: 相信這問題不少人都遇過
: 由於已經很難處理且不想處理了 所以正面迎擊這選項先刪除
: 此問題包含跨部門的工作協調和長官交辦任務
: 根據小弟的經驗 通常有以下四種應對方式
: 想要針對第四種方法和版上大大交換意見
: 1.裝死
  最好不要裝死
  碰到會黑人或陰人的長官 容易留下壞印象
  以後升遷會有問題

: 通常適用於不重要的工作任務，重要性的任務不適用
: 2.正面溝通
: 只適用於雙方都很理性 願意傾聽 而且工作氣氛和環境都良好的狀況下
 看你長官挺不挺你
 挺的可以正面溝通


: 3.離職
: 通常發生在忠厚老實的員工身上 或是真的很鳥的公司
: 結局就是被搞到受不了 換工作進入另一個火坑無窮迴圈 直到找到好工作為止

 離職很不好
 等於是你已經認輸了
 有很多時候 輸人不輸陣 應該要堅持下去

: 4.嘴砲話術唬爛
: 個人覺得第四種是最常見的 常見的伎倆有
:   1 想辦法賴到其他人身上
:   2 反過頭提出難題給對方 讓對方知難而退
:   3 拿比對方高階的長官來壓制他
:   4 找藉口搪塞
  嘴砲話術唬爛真的最常見
  簡單的講 出來工作 大家都是領一份薪水
  在不得罪人 不作奸犯科
  很多時候 都是把責任推到others
  但重點就是 推責任要推的有藝術
  讓人看不出來 或是根本不覺得你在推 還會覺得你很有道理 或是很合理
  這就很難哩




: 想針對第四點請益 有沒有更明確好用的方式或是具體經驗可以分享的
