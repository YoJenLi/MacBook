小弟工作資歷6年 主要是做PCB Layout (中間有跑去做HW1年)
產品有做過 Server PC NB

當初因為學歷不好 又想進大公司學東西 所以就傻傻去學了這個
重點:學PCB Layout沒有不好!!(但是只會餓不死)
當然一定有人會說 現在Layout都讓大陸人做了 在台灣的市場會越來越小
好的Layout 要看的東西很多
板廠 工廠 Power SI ME HW 各個環節都要顧到
Layout 在這中間要扮演的角色就是取得各部門的均衡點
有時候連RFQ 寫Proposal都要Layout來寫...
(因為往往各單位都會只注意自己想要的)
這就是我們需要做的事情 但是通常這就是吃力不討好
因為PM要求的只有出圖時間
而各單位要求的就是想要把東西改到好
領分紅的時候 support單位永遠是領最差的...TMD

或許有人會說Layout只是技術職而已
怎麼能跟HW比較呢!?
小弟在前年有轉作HW約1年的時間
發現台灣的系統廠HW好像並不是真正所謂的研發
劃線路的時候 老手交代 "照圖施工 保證成功"
然後有問題就找FAE 測試也可以凹FAE測試
(給想做FAE的人一句話 你們辛苦了 當然可能不是每一間公司都一樣^^)
HW 感覺做的事情很雜 感覺都是打雜的雜工

小弟目前的工作 算是部門內的小主管
底下帶了3-4個人
會的東西 也從以前只是會拉線
到現在整個板子評估 走線規劃 Power區域分布
SI高速線長度注意事項 ME組裝含Cable Routing
板廠工程問題事先預防 工廠打件問題預防
全方面的估計 把每次的工程問題 還有 打件問題 都控制在10項以下
但是還是沒用 因為只是Support單位...囧

薪水也從剛出社會的月薪21000 -> 26500 -> 37XXX -> 45000 -> 55000
年薪也從 50 左右慢慢的爬到了現在 85
相信要在爬上去因該有點困難 除非到了外商
奉勸各位剛出社會的新鮮人(肝)
請慎選工作 公司 至於PCB Layout 就排在最後的選項吧
因為即使在利害 能力再強 還是不會受到重用

到這邊一定有人會說 不爽不要做阿
對!!小弟目前就有不想做的打算
打算轉去當業務了...T.T
