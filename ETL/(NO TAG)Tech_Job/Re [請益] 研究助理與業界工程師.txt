一切都是看單位風氣、看老師好不好，還有你自己的命到底
如何，版上有很多大大分享過被凹的很慘的經驗了，之前在
這裡幫自家單 PO 徵人文也有被酸薪水太低。

那這次，我就來分享我自己覺得研究助理還不錯的原因唄，
我一定要跳出來說，也是有一切都照規矩來的老師的！

我自己是國立暨南大學資管碩士畢，學士也是在暨南唸，出
來工作經驗到目前是三年半，做的都是寫 code。

第一份工作是漢堡王做 Android 手機的 BSP 寫 Driver，因
為是新人，薪資也沒什麼好談的，就是 42K，但是自從進公
司後沒有在七點前下班過。

而在漢堡王最讓我不能理解的是，為什麼大家都要在上班的時
間逛 Mobile01、看網站、睡覺……然後等到下班時間了再來
開會和 debug 寫 code？家人住院所以我把事情做完準時下班，
準備要趕去醫院的時候，還被同事酸過怎麼那麼早走。=_=

半年後因為家人有一些狀況，所以就辭職轉到從家中走路就可以
到的研究院當研究助理，也是寫程式。

如同很多人講的，中研院就是依照國科會薪資起薪，碩士就是固
定 36K 起，然後之後會依年資調，我現在是三年年資，所以也調
到 41K 近 42K 了。

還是比華寶新人少沒錯，但我遇到的老師就是一切照規矩來的，進
來三年沒有所謂的被凹過，再加上自己待的小組的風格是大家開會
做事在上班時間認真做一做，早點下班回家吃飯，所以百分之九十
九都是準時上下班。

除了某些時候自己出包必須快點搞定，或是 debug 覺得就差那麼臨
門一腳所以不想走，三年來我沒有在六點半之後才下班過的經驗。

換句話說，我多了很多晚上時間做自己的事。

我可以學自己有興趣的新技術，可以寫自己的專案，可以有時間準備
投各 open source  的研討會的東西，沒動力的時候也可以打一整晚
的電動或看一整晚的小說。

而且這三年來我幾乎都可以在十一點前就準時就寢（除了打電動或寫
code 寫到忘我外）！

至於值不值得用每個月這 6K 以及以後加薪的幅度，來換以上的生活
品質，就見人見智囉。

不過我朋友說的好，在外面想加薪，實力比不上會和主管喇嘞重要，
也不是人人都能加到薪的。
