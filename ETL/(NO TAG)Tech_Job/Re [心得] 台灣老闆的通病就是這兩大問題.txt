※ 引述《corrona (弱酸性)》之銘言：
: 亂逛論壇看到這篇：「台灣老闆的通病就是這兩大問題，難怪有國際級競爭力的公司沒幾間」
: 裡面「三流老闆檢核表」我頂頭那隻就中7點！
: 先別講是不是尊重專業，不會把員工當奴工就很不錯了
: 無薪假如果是本世紀最偉大的發明，那「責任制」可以拿諾貝爾獎了
: 文章全文：
: 昨天我有個老朋友突然打電話跟我說，想換工作，問我APPWORKS大徵才裡面那些公司比較好比較值得去。
: 說真的，接到他的電話我還蠻驚訝的，因為他的工作薪水其實不差，而且在現在公司已經待了快十年，好歹也是個小主管的位置，理論上應該不會想換工作才對。詳細一問之下，才知道他老闆插手他負責的專案，把專案搞得亂七八糟，一個好端端原本大有機會的新產品，現在被搞成四不像，他實在受不了了，這次鐵了心就是想換工作。
: 我這朋友還算幸運，因為還沒結婚 (四十歲科技宅男，有人想應徵嗎?)，沒有養家活口壓力，換工作的風險對他來說還算承擔得起。但如果是一個已經成家立業的人，即使遇到這種老闆，恐怕都還是得吞下去。
: 老實說，台灣過去七八年的經濟慘況，雖然很多人把氣出在政府上，但台灣的老闆們恐怕要負上更大的責任。
: 如果我把身邊的朋友做個調查，會認為他現在的老闆是個好老闆的，恐怕不會超過 1/3。
: 台灣老闆的通病就是不尊重專業跟對員工小氣，難怪有國際級競爭力的公司沒幾間
: 1. 不尊重專業
: 這在台灣大小公司都很常看見，簡單來講，就是老闆要的最重要，其他一切專業意見都不重要。
: 工程師說 "這個案子一定要三個月才能開發完成喔"，不尊重專業的老闆的回答是 "不行，我要求你們一個月內一定得完成，否則就沒有年終獎金了。喔，對了，我還要加某某功能，但時程不能延後喔。"
: 業務說 "我們的產品價格太貴沒人要買"，不尊重專業的老闆就會說 "你看AAA公司的BBB產品就可以賣這個價格，我們的一定也可以(即使自己的產品其實爛很多)。"
: 以上場景有沒有似曾相似?
: 不尊重專業的老闆，基本上只會讓公司永遠成長不上去。
: 舉例來說，很多公司的老闆，明明沒有行銷專業，卻在公司花大錢拍廣告的時候，親自下來告訴廣告公司我覺得怎樣怎樣才對，那個鏡頭可不可以慢一點，這個演員的台詞我不喜歡，搞得好像自己是廣告專家一樣。最後廣告花大錢打了，但效果怎麼可能會好? 等於是把錢都丟到水裡去，這樣的公司怎麼可能成長呢?
: 2. 對員工小氣
: 台灣大多數的老闆的第二個大問題，就是太小氣，不管是不賺錢的公司或者是賺很多錢的公司都一樣。看看台灣的薪資被其他周邊國家海放就知道，連中國都追上來了。
: 當老闆不願意給員工好的薪資待遇的時候，員工要不就是工作打混，然後私下接副業賺錢，不然就是乾脆跳槽到海外去工作。而如果一間公司的好員工都跑掉，留下來的都是打混偷兼差的員工，這公司能夠好到哪裡?
: 其他國家的企業，把員工當成人才，但只有台灣的老闆們，把員工當成耗材。
: 我還看過某個老闆很得意地跟我炫耀說他公司的員工只是零件，他一點都不擔心任何一個人離職，新人只要訓練一下就馬上可以上手。
: 有這樣的老闆，怎麼可能會願意給員工好的薪水呢? 這種老闆腦中想的永遠只有，怎麼樣讓 "我自己" 賺最多，至於員工? 能少花錢就少花錢，最好是騙一些免費的實習生來做免錢的。
: 有這樣的老闆，台灣的企業怎麼可能會有國際化的競爭力呢?
: 這些慣老闆永遠喜歡大談狼性，但卻忘了，狼群是一起分享獵物，但慣老闆永遠想要獨佔成果。
: 台灣經濟環節裡面，最沒有國際競爭力的，其實並不是年輕人，而是台灣的老闆們。
