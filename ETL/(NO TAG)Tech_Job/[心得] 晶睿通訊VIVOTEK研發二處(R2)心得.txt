        各個處門因為帶頭的主管不一樣
        所以各有風氣XD
        看到樓上研發一處小伙伴爆炸
        小的私心想推一下敝公司研發二處
        不想讓大家覺得還待在敝公司的都是智障這樣XD

        因為自己興趣關係要轉換跑道
        不然實在沒有離開的理由
        有鑒於這間公司真的很好
        自己在這邊認識很多很棒的主管  很優秀的同事
        學到很多想學的技術
        所以特地來推薦一下這間低調好公司XD

        ＠公司名稱： 晶睿通訊 vivotek  (的研發二處(R2) )
                     因為公司有很多處
                     這邊只推我呆的研發二處( NVR team )
                     其他處我什麼都不知道呦 o'_'o

        ＠工作地點： 中和
                         地點就在公車站下車
                         以公車通勤來說算方便
                         但是那條路在挖捷運所以交通爛得徹底
                         最近的捷運站也有一小段距離
                         不過附近吃吃喝喝以宅宅需求來說算夠了


        ＠工時狀況： 基本上八小時做好做滿
                     8:30 - 17:30 是表定  RD 可以緩衝一小時
                     大部份的人 9:30 左右會到 18:30 左右會下班
                     可是誤差一點的話其實主管不太會在意
                     有時候有社團活動的同事會早點下班

                     基本上不太加班
                     18:30 有免費加班buffet
                     有一部份的人會吃完弄一弄再走
                     但忙的時候也是可以留到很晚
                     不過都不是強迫的
                     真的快不行了 call for help 主管跟資深都很願意下來幫你
                     沒有那種老鳥走之前菜鳥不準走的陋習
                     事實上我很菜的時候問過主管
                     我問他我每天都比較早走會不會對大家不好意思
                     主管說其實他根本不知道我幾點走也沒再管的  lol

                     另外就是有時候我做事拖拖拉拉太晚走
                     主管跟資深有時候會來關心你問你怎麼不回家

                     有次颱風天，部門主管還特定寫信請大家早點回家
                     我收東西收太慢六點多還在  還被部門主管拍肩關心
                     叫我注意安全早點回家QQ


        ＠工作內容：
                     基本上研發寫code，不會有雜事
                     除了DQA之外，研發二處還有自己的RDQA
                     尾牙也不用上去跳舞
                     進公司一年來我唯一覺得不想去的活動是hr辦的新人訓練

                     code的流程
                     RD用git上code ->
                       gerrit code review(同時CI(jenkins)+ 小小的UAT測試)

                     小妹親身經歷是  如果小妹的code寫太爛是不會被merge的
                     組長跟資深們都有認真把關 o'_'o
                     對於命名跟code的寫法也會有要求(大概是clean code那套)

                     技術方便包含上篇所說的
                     Twisted framework, Robot framework 什麼的
                     只要好用主管也不會限制你用
                     出去外面參加課程或是研討會也會有補助


        ＠工作環境:  座位是一般常見的隔板隔間座位
                     不小但也沒有特別大
                     公司會配電腦跟N個螢幕給你
                     因為是做NVR的所以螢幕有可能擺滿桌面
                     蠻特別的是因為是做IP cam 的公司
                     所以公司充滿著cameras XD
                     可以到處連來連去看公司的走道角落
                     有些人會把自己的camera放桌上
                     你連過去就可以偷看他(?  >//<
                     我座位前上方有個可以算高度的camera  蠻有趣的囧
                     也可以偷連出去看外面天氣
                     在充滿camera的環境工作，其實還蠻特別的就是了

                     另外就是關於吃的
                     公司有點心鐵櫃隨你吃
                     裡面的餅乾點心每天都會補一次
                     量多到一天吃不完
                     質量也不錯  小熊餅乾  義美  堅果  葡萄乾  大溪豆干都吃過
                     算是蠻有創意的
                     公司有研磨的咖啡無限量供應
                     愛喝茶的也有每天現泡的茶
                     冰箱的飲料也是無限量取用
                     我不太喝飲料的  不過大概就是可樂果汁各種茶飲

                     另外有蒸飯箱  微波爐  烤箱  都可以自由使用

                     公司有蠻好的投籃機
                     偶爾寫code累了同事們會一起去比投籃XD
                     有些同事就練就了雙手投籃的技巧
                     另外也有跑步機滑步機可以使用
                     也有浴室可以盥洗


        ＠主管人好不好： 這邊只針對研發二處(NVR team)
                         其他處我沒待過

                         組長都是萬中選一的
                         能力強不用說  基本上人都很好也很願意帶人
                         (ps 其實我不太加同事fb的
                         但是因為一個組長講話實在太好笑了所以我就加了)

                         部長們也都是好人
                         基本上不會有什麼勾心鬥角派系之類的事情
                         (主管們都是好朋友  頂多羽球廝殺xd)
                         特別要說的是研發二處的大頭
                         (處長還是協理的職稱我搞不清楚)
                         每一個月都會寫一篇很用心的研發二處公告
                         裡面會跟大家說這個月公司的營運  還有處上的發展等等的
                         我每個月都很期待也很認真看
                         算是很用心的一位"認真帶領團隊"的大主管

        ＠其他：
               這段算我自己加的
               VIVO的研發二處有自己的RD新人訓練
               個人覺得這段RD的新人訓練非常扎實
               (另外還會有hr那邊的新人訓練，不過相對起來蠻悲劇的就不提了)
               雖然剛開始會有點難適應
               但後來會覺得真是超棒的
               裡面的內容不是教你寫code
               而是把你從學生訓練成一個對自己以及對團隊負責的工程師
               內容大概就是 clean code 那套理論
               不過我這樣描述的話就太淺了  給有緣人自行體會:)
               訓練期40個工作天( 2~3 個月 ) 都是全薪  訓練期不用做工作
               會有一個很資深的資深全職訓練以及帶你
               (那位資深的工作就是訓練新人)

               薪水是VIVO常常被垢病的缺點
               但今年(2016)公司全面結構性調薪
               能談到多少就看個人本事了
               每一季都會有業績達成的獎金
               分紅跟年終都有一定水準
               至少小妹的研發二處給的算滿意XD
               不過 total 不能跟外商等級比就是了

               另外研發二處目前在跑 scrum
               該有的軟體開發流程也都有
               ( code review, CI, 版本控制, UAT)
               不能保證是給的最好的  但是以軟體工程師來說開發起來很舒服

               同事都是好爸爸好人等等
               菜鳥遇到困難通常都會有人跳下來教
               氣氛融洽
               下班後也都是好朋友

               旅遊補助會給一萬
               會有好多團(日本北越澳洲墾丁花東各種團)
               然後都還會有一次一天的秋遊免費參加

               關於工程師最擔心的PM問題
               沒有機歪的PM
               反而我們的PM都蠻為RD著想的
               而且又很專業  合作起來很舒服
               但是其他處的我就不知道了
