小弟我有幸拿到明年研替的Offer

面試其中在板上收穫良多，所以也來分享一下這個月來的面試心得


立琦科技

投影機/電腦: 皆有
筆試       : 無
部門/職缺  : ?/類比IC工程師
補助       : 單程高鐵票

當初是因為在內部工作的同學來跟我要履歷
所以才投看看，想說當作練功

不過HR通知我的時候聽到是面類比的缺
我聽到後是整個傻眼
但因為有補助單程高鐵票的關係
所以想說加減去練功看看

面試官是兩個工程師
我簡單做完自我介紹跟報告完碩班的研究內容後就結束了
也沒得到任何關於研究上的問題回饋

面試出來後有點傻眼
因為準備蠻多的，也有K蠻多關於類比的問題
不過他們聽完我研究內容後
也沒問其他類比相關的問題


結果: 三周後感謝函





瑞昱半導體

投影機/電腦: 皆無，建議自帶筆電
筆試       : 無
部門/職缺  : TV/數位IC工程師
補助       : 到高鐵台鐵或客運的單程計程車錢

狀況跟板上說的都差不多
一開始我用紙本投影片介紹自己的研究內容
因為我是做影像的，所以有得到蠻多研究內容的回饋
討論的過程中覺得，主管看起來很有興趣

研究內容大概討論一小時左右
就開始帶到白板題
問了基本的 clock skew、set up time、hold time
再來就問關於邏輯設計，邏輯閘相關的問題
還有3-tap的 FIR 怎麼做pippline去縮短critical path
最後的問題就是板上傳說中的邏輯考古題
我是被問到 "八顆球有一個跟其他不一樣種
在只有一個天秤的情況下，要怎樣才能在使用三次的情況下找出這顆球"
答案我就不說了，大家想看看~

問題問完後就開始介紹部門，並問我有沒有問題要問
我好像是問了關於新人訓練相關的問題
之後就是簡單的聊天，整體的面試我覺得蠻順利成功的


結果: 一周後人資打來說主管問我有沒有興趣加入
      結果三周後就寄感謝函給我，連二面都沒有QAQ




晨星半導體

投影機/電腦: 有些面談室有投影機，建議自帶筆電
筆試       : 考國文語意問題、數學排列組合、圖形邏輯問題
部門/職缺  : STB/數位IC工程師
補助       : 無

主管來之前會進行簡單的筆試
有些題目看起來也沒標準答案
而且主管感覺也沒有在意這筆試的成績如何
如果想要準備的話，建議看一看排列組合的高中題目就好

面試的過程也是自我介紹後
開始做研究內容的報告
主管一開始蠻認真聽的，也有問一些問題
後來發現我要報的東西蠻多的就問我要報多久
我立馬把後面我做的東西先做個總結
然後說大概再把這幾頁講完就OK了
之後也學到經驗，研究內容的投影片大概10~15頁最多
東西可以放擠一點沒關係，但是這樣可以幫助自己調整報告的時間
有些東西帶過也比較方便

研究內容說完了就開始白板題
一開始先問你會不會寫verilog
再出一個題目要你在白板上作答
我是被問到state machine，也有其他同學被問到gray code的題目
總之就是在白板上完整的寫出來
寫完後他也會依據你的code問問題
之後再請你把你的程式自己合成邏輯閘
還有如何簡化

再來就是部門介紹跟發問問題
我想說問題不要太普通，就問了個白目的問題
問說進去後是不是有機會會被換部門，還有三年後換領域會不會有困難之類的
主管回答我公司內調的機率很低，三年後想換領域你也贏不了該領域畢業的新人
總之就是問了個爛問題哈哈
整體的過程大概兩個小時

面試結束後大概一個禮拜內就接到二面通知
二面來的是另一個STB部門的主管
白板題因為上次就問過了，所以他也沒有多問
大概就是把研究內容講一遍
然後聊聊天，一個小時就結束了


結果: Offer Get
      一面後三天接到二面通知
      二面後兩個禮拜，HR通知錄取二面的部門



聯詠半導體

投影機/電腦: 皆有，但投影機很小
筆試       : 一些邏輯閘、C語言、verilog的問題，算蠻基本的
部門/職缺  : Driver、TV、小面板IC/數位IC工程師
補助       : 台南新竹高鐵票一張

面試前剛好接到晨星Offer
所以面試完後有馬上問聯詠HR有沒有錄取

再主管進來前會有簡單的筆試
題型大概是我上面列的那樣，但題目應該一陣子一陣子會變
最近有要面試的可以私訊我
考試時間大概40分鐘，面試的時候主管也會看你筆試的回答如何

面試是一次面三個主管
前面也是自我介紹加研究內容報告
因為面很多次了，所以這次即使三個主管也是蠻順利的

面完後他們就開始看你履歷發問
問一些修課相關的問題 (面這麼多次就這次被問最多)
還有自傳跟成績單
成績單被盯蠻慘的，不過也是因為我大學跟碩班成績真的蠻爛的
甚至還被問我大學成績這麼爛是怎麼正取35進研究所的
因為沒有預設到會被盯這麼慘，所以回答的不是很好

大概面一小時後就開始自由發問
然後一次三個部門真的很難問問題
我是靠他們各自介紹完後，馬上提出關於該部門相關的問題
不然等他們一一介紹完後，你會真的不知道問甚麼
面試結束後就人資進來，然後給我一張高鐵兌換券


結果: 因為希望人資馬上給我回復所以當晚就被發感謝函了 Orz
      但成績被盯成這樣其實也不太意外
      聯詠面試好像都是面完後主管馬上決定
      所以面完後人資都會跟你說有沒有上，而通常有拿到其他伴手禮的
      就代表有上XDD



本來還有打算面創意電子 跟台北奇景光電
但因為已經答應小m所以就都推掉了

去面聯詠的時候有跟人資聊一下
聽人資說聯詠面前研替還有很多缺額
(我聽到的是140還是40個缺只找到20個，不知道有沒有聽錯)

所以還沒收到聯詠面試的不要氣餒，還有機會!
(不過他跟我講完後我就被寄感謝函了，有點被打臉QAQ)

有問題也可以私訊我
如果我知道的我都可以幫忙，希望還沒找到研替的也能順利的找到工作
