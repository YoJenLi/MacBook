以下觀點都是對於自己面試的部門 / 廠區 的 主觀意識

很有可能以偏概全  單純分享  但實際還是請版友自行判斷囉

1.以上三家公司都是HR招募看104看到的 所以104上的履歷還是請認真填寫

2.面試前準備 :

  基本的公司初步了解 包含 面試職缺工作內容 公司文化  基本技術

  自我介紹(中/英文) ; 模擬HR / 主管的問題 ; 準備要問HR / 主管的問題

  其他都是臨場反應囉

3.面試可觀察的重點 :

  A. 門面 : 包含Lobby / 警衛 / 內部職員 / HR 的穿著打扮談吐

     這很膚淺 但是很準 , 給我的感覺是 台G 略優 台達電 >>>>> 日月光 (中壢)

  B. 考試內容 : 包含英文 / 邏輯智力測驗 / 專業 / 以及最重要的 HR 或主管如何
                看待這份成績

  C. 面試的提問 : 如果被問到的問題都是網路上查到的也可以感受得出來主考官
                  相對應的用心在意程度

  D. 自我的優點以及缺點

  E. 經過面試後 慢慢可發現自己適合的職缺特性

3. 結論特色 :

   台積電要的是忠誠的新鮮肝 ; 錢最多 第一年約 90~100

   台達電要的是忠誠好教的員工 ; 錢一般 感覺有點公務員 ( ? )

   日月光要的是奴才 ; 開出的薪水給HR看到後就面有難色 沒見到主管就被發掰掰卡了
