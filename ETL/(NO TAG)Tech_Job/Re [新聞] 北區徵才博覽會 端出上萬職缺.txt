: 北區徵才博覽會 端出上萬職缺
: http://tinyurl.com/9bsucd2
: 中國時報【陳宥臻╱台北報導】
: 救失業，經濟部端出上萬名職缺，九月八日(下周六)舉行北區產業徵才博覽會，地點在台
: 北展演二館（原世貿二館），僅舉辦一天，職缺以民生服務業占比五成以上最多，其次是
: 科技製造業和綠色能源產業。參與的知名業者包括晶元光電、友達光電、王品餐飲、美食
: 達人（85度Ｃ）等。
: 經濟部能源局表示，目前有近一三○個公司參與徵才，將釋出一萬個以上職缺，其中民生
: 服務業務占五十四％，科技製造業三十％和綠色能源產業占十六％。
: 由於經濟部今年已在高雄、台中舉辦過兩場徵才博覽會，共釋出一萬七千多個職缺，台北
: 場是今年最後一場。
: 這次參與廠商包括綠色能源產業的晶元光電、友達光電、綠能科技、光陽工業等；科技製
: 造產業的台灣玻璃、震旦集團、中國製藥、神腦國際、緯創軟體等；民生服務產業的統一
: 超商、全家超商、台灣麥當勞、王品餐飲、美食達人（85度Ｃ）、台灣高鐵等。職缺內容
: ，包括工程師類、管理師、作業員、技術員、服務員、店長、 儲備幹部等。
: 參與徵才博覽會的一一一一人力銀行表示，多數的職缺都是面議薪資，目前釋出最多人力
: 的是麥當勞，有一千名職缺，內容為門市人員和儲備幹部。其他如永慶房產集團則明確指
: 出，保障新人前九個月保障月薪為五萬元，比很多新鮮人的薪資行情高一倍。
: 能源局副局長王運銘表示，活動雖然只有一天，但為提升求職者的職能競爭力，活動當天
: 同時規劃職涯講座，內容含面試禮儀、工作職能探索、求職核心競爭力、撰寫優質履歷等
: ，由專家提供求職者意見及協助。配合這項徵才活動，市政府捷運站備有免費接駁專車。
